Avinya – Web Platform

Avinya is a dynamic web platform built using PHP and MySQL that enables users to manage events and content efficiently. It supports multiple users, allowing seamless interaction between frontend and backend systems. The platform focuses on usability, structured data handling, and smooth user experience.

Tech Used: PHP, MySQL, HTML, CSS
