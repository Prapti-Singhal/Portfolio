License Plate Detection System

A computer vision-based system that detects and reads vehicle license plates from images. The system uses Haar Cascade for plate detection and histogram-based segmentation for extracting characters, followed by text recognition. It is designed to handle multiple input conditions and provides results along with error handling for unclear images.

Tech Used: Python, OpenCV, Haar Cascade, Histogram-Based Segmentation, Image Processing
