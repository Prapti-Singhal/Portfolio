Secure Chat Application (RSA)

A secure messaging system built using RSA cryptography to ensure safe communication. It implements encryption and decryption mechanisms along with secure key exchange to protect user messages from unauthorized access.

Tech Used: C++, RSA Algorithm, Socket Programming
